# 🚀 Installation et Lancement

## Prérequis
- Node.js 16+
- npm ou yarn
- Git

## Installation

### 1. Cloner le repo
\\\ash
git clone https://github.com/[votre-username]/SyllabeKids
cd SyllabeKids
\\\

### 2. Installer les dépendances
\\\ash
npm install
\\\

### 3. Lancer en développement
\\\ash
npm run dev
\\\

L'app est accessible sur http://localhost:5173

## Build Production
\\\ash
npm run build
npm run preview
\\\

Visite http://localhost:4173 pour tester la version production.
